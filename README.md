# Installation

```
npm install
```

# Configuration

In example/my_server.js, provide the AndroidTV IP:

```js
let host = "192.168.1.25";
```

# Usage

```bash
cd ./example
node my_server.js
```

Open the browser at: http://127.0.0.1:3000/remote.html
